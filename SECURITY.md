# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.1.x   | :white_check_mark: |
| < 1.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT open a public issue**
2. Email: **security@denomas.com** (or use [GitHub Security Advisories](https://github.com/Denomas/freemind-ce/security/advisories/new))
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Affected version(s)
   - Potential impact

We will acknowledge your report within **48 hours** and provide a fix timeline within **7 days**.

## Scope

- FreeMind CE application code
- Build and CI/CD pipeline
- Dependencies (transitive included)

## Out of Scope

- Original FreeMind (SourceForge) — report upstream
- Social engineering attacks
- Denial of service against GitHub infrastructure
